# Projeto Integrado de Engenharia de Software - Sistema Desktop - AmparoEdu
O presente projeto foi desenvolvido no ano de 2025 por acadêmicos da Universidade Federal do Ceará, como parte da construção prática da disciplina de Projeto Integrado de Engenharia de Software, com o intuito de atender às demandas digitais da ONG APAPEQ no que tange o campo de Atendimento Educacional Especializado.
